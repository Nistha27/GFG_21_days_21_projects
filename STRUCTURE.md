- Each day folder contains both tutorial and assignment notebooks

- Model weights are included for Days 8-9
- Large files have been compressed (e.g., Resume.csv.zip)
- Environment files (.env) are excluded; only templates (.env.example) are included
- Jupyter checkpoint folders (.ipynb_checkpoints) are kept for backup purposes
